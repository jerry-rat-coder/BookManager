# BookManager
前后端分离图书管理系统项目

## 技术栈

Vue2+vue-router+vuex+Axios+Element-UI+echarts

## 功能说明

图书管理系统，主要实现了借阅和管理图书的增删改查功能。图书管理员可以进行图书的入库、出库、借还、查询等操作，同时还可以对用户信息进行管理。普通用户可以浏览图书信息、搜索图书、借阅图书等操作。
